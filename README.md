Install packages
```
composer install
```

Copy .env file
```
cp .env.example .env
```

Set database detail
```
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=
DB_USERNAME=
DB_PASSWORD=
```

Start laravel project
```
php artisan serve
```
